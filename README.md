# Clínica de Fonoaudiologia FonoVida - Dra. Mariana

Uma landing page moderna, responsiva e acolhedora desenvolvida para a Clínica de Fonoaudiologia FonoVida, liderada pela Dra. Mariana. O projeto foca em transmitir confiança, profissionalismo e humanização no atendimento fonoaudiológico.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível.
- **CSS3 (Vanilla)**: Design personalizado, sistema de cores harmonioso, animações suaves e layout responsivo.
- **Phosphor Icons**: Conjunto de ícones modernos e minimalistas.
- **Google Fonts (Outfit)**: Tipografia premium para melhor legibilidade.

## ✨ Funcionalidades

- **Design Responsivo**: Adaptado para dispositivos móveis, tablets e desktops.
- **Seção Hero**: Chamada para ação (CTA) clara e impacto visual.
- **Serviços/Especialidades**: Detalhamento dos tratamentos oferecidos (Linguagem Infantil, Voz Profissional, Reabilitação Auditiva).
- **Sobre a Especialista**: Biografia profissional com foco em autoridade e empatia.
- **Depoimentos**: Prova social para gerar confiança nos novos pacientes.
- **Integração com WhatsApp**: Botão direto para agendamento rápido.
- **Rodapé Completo**: Informações de contato, localização e redes sociais.

## 📁 Estrutura do Projeto

- `index.html`: Arquivo principal contendo a estrutura da página.
- `style.css`: Estilização completa, incluindo variáveis de design e responsividade.
- `assets/`: Imagens e recursos visuais utilizados no projeto.
  - `hero-fono.png`: Imagem principal da seção de destaque.
  - `dra-mariana.png`: Foto da especialista.

## 🛠️ Como executar localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/TFS-Data/clinicafonovida.git
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd clinicafonovida
   ```
3. Abra o arquivo `index.html` em seu navegador de preferência.
